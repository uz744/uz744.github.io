## 🚀 How to Update Your Portfolio

### Adding New Skills
1. Open `index.html` in a text editor
2. Find the "Skills Section" (around line 95)
3. Copy a skill-card div and modify it with your new skill

### Changing Colors
1. Open `style.css`
2. Find the `:root` section at the top
3. Modify the color variables

### Adding Projects
1. You can add a new "Projects" section in `index.html`
2. Use the same card structure as the skills section

### Deploying Changes
1. Make changes to the files
2. Commit to GitHub
3. Your website will update automatically!
